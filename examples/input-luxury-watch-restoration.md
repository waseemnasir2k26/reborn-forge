## Why this works for the same generator

The same META-PROMPT that generates a mansion master prompt also generates a watch master prompt — because the structural pattern is identical:

| Layer | Mansion | Watch |
| --- | --- | --- |
| Hook stage | filthy room reveal | dirty watch macro on workbench |
| Cleaning stage | scrub / wash / strip | ultrasonic / brush |
| Build stage | walls / ceiling / floor | movement service / dial work |
| Reveal stage | epoxy floor pour | case polish + on-wrist |
| Payoff | golden-hour final living room | wrist shot under warm light |

The Pattern Engine agent extracts this isomorphism. The Prompt Architect agent locks the niche-specific specifics (audio = ultrasonic vs hammer; lighting = clinical vs warm; lens = macro vs wide).

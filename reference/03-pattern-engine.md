# Agent 3 — Pattern Engine

> Converts the analyzed video into a reusable system. Specifics removed; logic preserved.

**INPUT:**
- NICHE PROFILE: `01-niche-intelligence.md`
- VIDEO ANALYSIS: `02-video-analysis.md`

---

## [CORE PATTERN]

**Cold Dread → Origin → Spread → Collapse → Aftermath → Echo.**

Each step has a single function:

| Step | Function | What changes |
|------|----------|--------------|
| Cold Dread | Hook + tone-set | Viewer commits to ASMR register |
| Origin | Anchor + cause | Viewer knows when, where, why |
| Spread | Escalation | Stakes compound — each city worse than the last |
| Collapse | Climax | Imagery saturates; music peaks; mortality numbers maximize |
| Aftermath | De-escalation | Society survives, transformed |
| Echo | Coda + tease | Viewer is delivered to next video |

This pattern is **invariant** across the genre. Any topic — plague, war, eruption, fire, fall of empire — maps to the six stages.

## [STRUCTURE MODEL]

**6 sections. Fixed order. Hard percentage targets.**

| Section | % runtime | Min duration (8 min video) | Max duration (18 min video) |
|---------|-----------|----------------------------|------------------------------|
| 1. Cold Dread | 0–5% | 24 sec | 54 sec |
| 2. Origin | 5–25% | 1.6 min | 3.6 min |
| 3. Spread | 25–50% | 2.0 min | 4.5 min |
| 4. Collapse | 50–75% | 2.0 min | 4.5 min |
| 5. Aftermath | 75–92% | 1.4 min | 3.1 min |
| 6. Echo | 92–100% | 0.6 min | 1.4 min |

Scene count = **3–4 scenes per minute** of runtime (longform) → 24–32 scenes in 8 min, 42–48 in 12 min, 54–72 in 18 min.

For 9:16 Short: collapse to **5 sections** (Hook → Wonder → Proof → Mystery → CTA), 15–20 scenes, 1.5–2.5 sec/scene.

## [RETENTION MECHANISM]

**Layered retention** — each layer compounds.

1. **Atmospheric trance** — desaturated grimdark + 60–75 BPM dark-ambient + 4–6 sec shots → viewer enters a near-meditative state. Bounce rate drops because scrolling feels jarring.
2. **Numerical escalation** — mortality stats climb monotonically across Spread → Collapse. Brain tracks the climb and waits for the peak.
3. **Named-individual zoom** — Origin and Spread sections drop a named historical figure every 60–90 sec. Brain anchors to faces; faces become hooks.
4. **Anchored expectation gaps** — date + place anchors at scene-openers create micro-expectations ("what happened in Avignon next?"). Each scene answers one and creates one.
5. **Quiet payoff at end** — Echo section is deliberately quiet, almost relieving. Viewer's emotional release becomes the reward, which trains them to return.

## [PROGRESSION LOGIC]

**Each section changes one variable. Never multiple at once.**

- **Color saturation:** 55% (Cold Dread) → 50% (Origin–Collapse) → 60% (Aftermath) → 65% (Echo, slight warm sunrise).
- **Music density:** silence (Cold Dread) → drone (Origin) → drone + cello (Spread) → drone + cello + choir (Collapse peak) → drone (Aftermath) → silence + bell (Echo).
- **Mortality on screen:** 0 (Cold Dread implicit) → 1–10 named (Origin) → hundreds (Spread) → thousands (Collapse) → echoed retrospect (Aftermath) → 0 (Echo).
- **Architectural decay:** clean (Origin) → crowded with bodies (Spread) → bodies + abandoned (Collapse) → ruined-and-rebuilt (Aftermath) → still ruin, calm light (Echo).
- **Weather:** overcast (Cold Dread) → fog (Origin) → rain (Spread) → snow / dusk (Collapse) → dawn fog (Aftermath) → sunrise (Echo).

**Rule:** Scene N+1 = Scene N + ONE change. Never change position + action + weather all at once. Continuity requires single-axis evolution.

## [REPEATABLE TEMPLATE]

Generic structure, reusable across any historical event:

```
[1. COLD DREAD — 0–5%]
- Single grim image of {symbolic-object | symbolic-aftermath}
- Whispered VO: "{In year X, in place Y, {visceral physical event} — and no one knew why}"
- Silence + single SFX (bell / cough / scream / wind)

[2. ORIGIN — 5–25%]
- Date + place anchor
- Causal chain (verified, era-locked, cited)
- Patient zero / vector / inciting moment
- 1–2 named historical figures introduced
- Music enters at −22 LUFS, drone in minor key

[3. SPREAD — 25–50%]
- City-by-city / region-by-region escalation
- Mortality counter climbs (on-screen text)
- Recurring witness-character or POV-narrator threads
- Music adds cello, sustained pad

[4. COLLAPSE — 50–75%]
- Iconic peak imagery (mass grave / burning city / battle aftermath)
- Highest mortality numbers
- Quote from period chronicler
- Music peak: −18 LUFS, choir enters

[5. AFTERMATH — 75–92%]
- Society broken + remade
- Long-term consequence (labor shortage / religious shift / political collapse)
- Slight warm color reintroduction
- Music falls back to drone

[6. ECHO + CTA — 92–100%]
- Coda line: "The {world | empire | city} that {emerged | survived | remembered} was not the same."
- Silence, single bell, fade
- Soft CTA: "If you want more forgotten history brought back to life — subscribe."
- Next-episode tease
```

The template is **language-agnostic** and **subject-agnostic** within the grimdark history niche. Plug in:

- Subject (plague / war / eruption / fire / siege / famine)
- Era (antiquity / medieval / early modern / industrial / 20th-century)
- Geography (Europe / Mediterranean / Middle East / Asia / Americas)
- POV (witness / scholar / ruler / child / map-narrator)
- Visual motif (empty street / mass grave / doctor-priest / ship arrival / burning city)

→ produces a unique video that obeys the genre's invariants.

## Why this pattern works

It is **fractal**. The 6-stage arc plays at the macro (12-min video), repeats at the meso (each 2-min section internally has its own micro-cold-dread → micro-aftermath), and even at the micro (each individual scene opens with stillness, peaks at action, holds, releases). The viewer's brain is being trained on the same shape three times in 12 minutes — which is why the format scales to a series of 50+ episodes without fatigue.

# 🛠️ Production SOP

1. Paste master prompt → INPUT → Phase 1 script.
2. Save `SCRIPT-<topic>-SHORT.md` → `python _build.py` → PDF.
3. `phase 2` → image prompts → `IMAGE-PROMPTS-<topic>-SHORT.md`.
4. Generate 9:16 stills in Nano Banana, 2–3 variants/scene.
5. `phase 3` → JSONs → `SCENES-<topic>-SHORT.json`.
6. Render 2–4s clips in VEO3 / Sora 2 / Runway Gen-4 (9:16).
7. CapCut: assemble vertical, ElevenLabs Hindi VO, music bed −15 dB, snappy cuts, on-screen Hindi text.
8. QA Failsafe.
9. Publish to YouTube Shorts + IG Reels + TikTok cross-post.
